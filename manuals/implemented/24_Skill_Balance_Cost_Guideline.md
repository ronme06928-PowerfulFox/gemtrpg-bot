# スキル威力・使用コスト・取得コスト調整メモ

**種別**: implemented（2026-06-28 確定）

この文書はスキル調整基準の確定版である。スプレッドシート反映済み。今後は実プレイで問題が出た箇所を差し戻して調整する。

2026-06-26時点で、以下の基準は暫定固定とする。今後はこの基準を前提にキャッシュ上の現存スキルを確認し、実プレイやテスト戦闘で問題が出た箇所を差し戻して調整する。

## 1. 基本前提

- 標準戦闘は8ラウンド程度を想定する。
- 味方HPはおおむね120程度を想定する。
- 敵HPは50〜60程度の小型から、数百程度の強敵・ボスまで幅を持たせる。
- 能力補正は一旦+4で統一して考え、スキル固有威力の段階評価からは除外する。
- 基本技は最低限の攻防手段であり、単なるリソース回復行動ではない。

### 戦闘テンポと敵編成

敵の想定撃破ラウンドは、単体のHPだけではなくエンカウント全体で見る。

| 敵区分 | 想定 |
|---|---|
| 雑魚 | 複数体で出現し、戦闘全体で4ラウンド程度 |
| 中堅 | 1～2体、または雑魚より重い編成で8ラウンド程度 |
| 強敵・ボス | 10ラウンド以上。HPだけでなく段階変化や行動パターンで長さを作る |

雑魚は個体HP50～60程度でも、複数体で出るなら総HPは150～180程度になり得る。

範囲・複数対象攻撃の詳細基準は一旦保留する。現状、味方側が使うことは想定しない。敵側や将来の追加スキルで扱う場合は、雑魚戦で強くなりやすい点を踏まえ、単体高威力スキルと同じ単体効率にならないようにする。

### 敵火力・回復・防御

敵火力は、プレイヤー側が無抵抗なら5ラウンド程度で危険域に入る水準を目安にする。味方HP120を基準に、無抵抗時の被害が1人あたり累計100前後に届き得る程度を上限目安とする。ただし、実戦ではマッチ、防御、回避、対象分散、敵撃破による手数減少が入るため、常時そのまま通る前提では見ない。

回復は控えめに扱う。戦闘テンポを伸ばしやすいため、味方側に多く持たせることは基本方針としない。回復スキルを置く場合も、同コストの攻撃や防御より明確に安全な選択になりすぎないよう、量・条件・使用回数・リソース負担で抑える。

防御・回避などの守備系スキルはやや軽めに評価する。マッチに勝っても相手へダメージを与えないため、同じ威力・同じコストの攻撃スキルより取得コストや使用コストを軽く見てよい。

### ボス段階上昇

ボスの段階上昇は混合型を基本案とする。ターン経過、HP割合、固有ギミックのいずれか、または複数条件を組み合わせて段階を進める。具体的な進行条件はボスごとに個別設計する。

## 2. 威力段階

スキル固有威力の中心値は以下を基準とする。

| 段階 | 役割 | 中心威力 |
|---|---|---:|
| 1 | スターター・軽い主力 | 8 |
| 2 | 繋ぎ・お膳立て・主力 | 11 |
| 3 | 大技・決定打 | 15 |

段階間にはおおむね2〜4程度の威力差を持たせる。条件達成時に一段階上へ届くスキルは許容する。ただし、条件を満たさない通常時の性能、使用コスト、取得コスト、準備負担を合わせて評価する。

## 3. 基本技と機会費用

基本技のリソース獲得を評価に含める。

- 物理基本技：使用後FP+2
- 魔法基本技：使用後FP+1、MP+1

したがって、表示コストだけではなく、基本技を使わなかった機会費用を含める。

```text
物理スキルの実質FP負担
= 表示FPコスト + 基本物理で得られたはずのFP2

魔法スキルの実質負担
= 表示FPコスト + 基本魔法で得られたはずのFP1
= 表示MPコスト + 基本魔法で得られたはずのMP1
```

このため、表示FP1の物理攻撃は単純に「軽すぎる」とは判定しない。基本技との差分ではFP3相当を払っている。

## 4. 使用コストの暫定基準

標準戦闘8ラウンド、初手基本技、マッチ由来FPは標準で3点程度として見る。下振れ検算では0〜2点、上振れ検算では4点程度を見る。

| 区分 | 目安 |
|---|---|
| 物理 段階1 | FP1前後 |
| 物理 段階2 | FP3前後 |
| 物理 段階3 | FP5前後 |
| 魔法 段階1 | MP2 / FP0前後 |
| 魔法 段階2 | MP3 / FP1〜2前後 |
| 魔法 段階3 | MP6 / FP3前後 |

これは絶対値ではない。次の要素で上下する。

- 的中時効果か、使用時効果か。
- 即時効果か、次ラウンド以降の効果か。
- 単体で完結するか、後続行動が必要か。
- 状態異常やバフが即効性を持つか、蓄積・消費して初めて価値を持つか。
- 副作用や対象選択制限があるか。
- 基本技の機会費用がどれだけ重いか。

## 5. 状態異常・追加効果の評価

同じ数値でも、直接ダメージより状態異常付与を高く評価する。ただし、状態異常付与だから即座に高コストへ寄せるのではなく、成立条件と実効性を見る。

- 直接ダメージ：即効性はあるが展開価値は低め。
- 出血：状態異常付与の標準寄り。出血4〜5程度の的中時付与は、取得1スターターの標準範囲として扱う。
- 破裂：破裂爆発で起動して初めて直接ダメージになる。起動手段が必要なため、他の状態異常より付与数値はやや大きめに取る。
- 亀裂：的中ごとに追加ダメージを与えるため、攻撃回数が増えるほどリターンが膨れやすい。継続ラウンド、ラウンド付与制限、低めの付与数値で抑える。
- 減速・挑発・被ダメージ増加：対象選択や行動順、集中攻撃に影響するため高価値。ただし副作用や遅延があれば相殺する。
- 蓄力・凝魔などの自己蓄積：後続で消費して初めて価値が出るため、即時効果より軽く見る。
- 再使用・踏み倒し・変換・遷延：ビルドの動きを変えるため高価値。ただし低威力、成立条件、追加消費、準備要求があれば取得1〜2に置ける。

## 6. 取得コストの考え方

取得コストはスキル段階と機械的に連動させない。

新規作成時の取得Ptは平均8Pt程度、最低寄りで5Pt、最大10Pt程度を想定する。基本的な1ビルドは6Pt程度で成立し、余りで有用な他スキルを1〜2個取得できる形を目指す。

| 取得コスト | 意味 |
|---:|---|
| 0 | 基本技・共通前提 |
| 1 | ビルドの基本パーツ |
| 2 | 主力・中核・初動上位互換・強い展開札 |
| 3 | 希少な個性化枠 |
| 4以上 | 原則例外。特殊取得・上位解放・成長後向け |

### 取得1で許容しやすいもの

- 段階1程度の威力と的中時状態異常。
- 表示コストは軽く見えるが、基本技の機会費用込みでは相応に重いスキル。
- 後続行動が必要なお膳立て。
- 蓄力・凝魔など、後で消費して初めて価値が出る準備札。
- 副作用つきの強効果。
- 基本技の完全上位互換ではない初動札。
- 複数の準備札、自己リスク、重い使用コストを要求する条件付き大技。

### 取得2に寄せるもの

- 初動から安定して強い。
- 基本技の上位互換に近い。
- 高威力と強い状態異常を同時に持つ。
- 命中不要で明確なメリットのみを得る。
- 単体で完結する強効果。
- ビルドの主要な勝ち筋を作る。
- キーワード越境や、パーティー内役割分散を可能にする。

### 取得3に寄せるもの

- 既存ビルドの質を変える。
- コンボを大きく拡張する。
- 強い持続、変換、踏み倒し、放出、遷延を持つ。
- 「放出の呪い」のような希少な個性化枠。
- サービス序盤の通常大技ではなく、後の成長段階・上位水準の技。

## 7. 横断的な判断基準

### 守備系スキル

防御・回避スキルは、マッチに勝利しても基本的には相手へダメージを与えない。そのため、同じ威力・同じリソース獲得効果でも、攻撃スキルよりやや軽めに評価する。

### 属性タグの差別化

斬撃・打撃・貫通ごとに、スキル単体の数値や状態異常量には多少の差があってよい。今後は出血・破裂・亀裂のような状態異常だけでなく、属性タグそのものを強化するパッシブや効果を用意し、それぞれの属性を選ぶ理由を作る。

### 自傷・自己リスク

自傷や自己出血は、単なる軽いコストとして扱わない。HPコスト、自己出血による継続ダメージ、次ラウンド以降に抱えるリスクを含めて評価する。

### 複合キーワード・ロマン技

複数キーワードを要求するスキルは、表示取得コストだけでなく、準備スキルの取得、パーティー協力、戦闘中の状態蓄積を含めて評価する。条件達成が重い場合、派手な効果を持っていても取得1〜2で許容できる。

### セット運用

特定のスキルを本領発揮させるために別スキルが必要な場合、単体の取得コストだけでなくセット全体の取得コストで見る。

## 11. Codexスキルへの反映方針

現時点では、Codex用の `battle-balance-designer` スキル本体へ直接反映しない。理由は以下。

- この文書はまだ調整中のドラフトである。
- スプレッドシート修正後のキャッシュ再取得・差分確認は一部完了しているが、まだドラフト全体の確定レビュー前である。
- 実スキルはリポジトリ外のCodex設定領域にあり、編集は通常のリポジトリ作業より影響範囲が広い。

反映する場合は、以下の要点をスキル側の評価手順に追加する。

- 基本技のFP/MP獲得を機会費用として評価する。
- 取得コストはスキル段階と機械的に連動させない。
- 取得1/2/3は「初期ビルド成立性」「中核性」「希少な個性化」で判定する。
- 状態異常は直接ダメージより高く見るが、的中条件・遅延・副作用・準備要求で相殺する。
- 破裂・亀裂・出血は同じ数値でも価値が異なる。
- 複数キーワード、自己リスク、パーティー協力、セット運用を評価に含める。
- 守備系は勝利してもダメージを与えないため、攻撃札よりやや軽めに評価する。

スキル本体への反映は、キャッシュ更新後の差分確認と、追加調整が一段落した後に行う。
