# スキル調整案表

基準文書は [24_Skill_Balance_Cost_Guideline_Draft.md](./24_Skill_Balance_Cost_Guideline_Draft.md) を参照する。

## 2. 全体結論

現時点で、スプレッドシートへ即時反映すべき明確な修正候補は少ない。

既に議論済みの `Pb-13`、`Pp-13`、`Ps-13`、`Mb-06`、`Mp-08`、`Pp-10` はキャッシュ上で修正・確認済みであり、現行値を前提にしてよい。

今後の実プレイで見るべき主な観点は以下。

- 取得3スキルが、初期作成時のビルド幅を狭めすぎないか。
- 回復・MP回復系が、魔法キャラのリソース制約を崩しすぎないか。
- 準備型・ロマン型が、雑魚4R戦闘で遅すぎないか。
- 中堅8R、強敵・ボス10R以上で、状態異常ビルドが本領を発揮できるか。

## 5. 調整時の運用ルール

スプレッドシートを修正する場合、次の順で判断する。

1. まず使用コストを調整する。
2. 次に状態異常量・継続ラウンド・成立条件を調整する。
3. それでも役割が過剰なら取得コストを調整する。
4. 取得3以上は、通常の強スキルではなく、ビルドの質を変える希少枠に限定する。

現時点では、取得コストの大規模な再配分は行わない。
