JSON定義ビルダー（rule_data / v2 strict）

現行仕様（skill_json_rule_v2 strict）向け。schema必須・buff_id中心でJSONを生成します。

基本設定

初心者モードでは必要度の低い入力を折りたたみ表示します。

自然言語入力（3列）

運用列: 使用時効果 / 発動時効果 / 特記。`的中時`を使用してください（`中時`/`命中時`はエラー）。

ここで入力したタグは基本設定のtagsへ反映されます。

power_bonus

2. レシピ集（目的別サンプル）
3. 完全仕様（運用ルール）
・`apply_to`: 未指定は `base` として扱う。
・`max_bonus` / `min_bonus`: 計算後の補正値を上限・下限で丸める。
・`condition` は `source/param/operator/value` の4項目が揃って初めて有効。
・`CONTAINS` は tags など配列判定向け。数値判定は `GTE/LTE/GT/LT/EQUALS` を使う。

effects

FP2/MP3消費・WINで震盪2スタック(3R)・HITで破裂+3 を一括入力します。
